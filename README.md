Код по тестовому заданию. 

В папке `data_parsers` лежат скрипты для создания json файлов с путями к картинкам, маскам и промптами для датасетов DIS, BIG и ThinObjects

В ноутбуках пример теста на видео (и визуализации) и пример запуска и расчета IoU на датасете BIG.

В `datasets/sequences` - последовательности картинок для теста на видео.

Остальное - скрипты с вспомогательными функциями.

Важно - для инференса на видео я использовала модифицированные версии скриптов (для выгрузки на cpu), которые можно скачать в комментариях здесь https://github.com/facebookresearch/sam3/issues/169
